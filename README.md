
基于 next.js + antd + dva 的简单例子。

# Usage

## 1、Clone project code.
```
git clone git@github.com:chihm/next-with-antd-dva.git
cd next-with-antd-dva
```

## 2、Installation dependence.
```
yarn install
```

### Or
```
npm install
```

## 3、Start local server.
```
yarn run dev
```

### Or
```
npm run dev
```
